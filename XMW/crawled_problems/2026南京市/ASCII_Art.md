# ASCII Art

> 时空限制：1秒 / 128MB

## 题目描述

早期计算机的输入设备 “终端” 由打字机演变而来，只能显示字符。在字符终端上画画被称为 ASCII Art (ASCII 艺术)。

![图片](/file/2026/05/20260511024951_T4-1.png)

我们希望用 ASCII Art 更 “酷” 地绘制迷宫。迷宫是一个 $n$ 行 $m$ 列的矩形。迷宫中，障碍格用 “#” 表示，空格用 “.” 表示。这个迷宫还满足 “无环” 性质：任意两个空格之间恰好只有一条路径。

我们希望绘制的迷宫能正确还原迷宫中空格的连通关系。

## 输入格式

输入第一行两个整数 $n, m$，表示迷宫的行数和列数。
接下来 $n$ 行，每行一个长度为 $m$ 的字符串，仅由 “.” 和 “#” 组成。
字符 “.” 表示空格，字符 “#” 表示障碍格。

## 输出格式

输出 $2n+1$ 行，为迷宫对应的 ASCII Art 图形。

## 输入输出样例

```input
4 6
...#..
##...#
...#..
#.#..#
```

```output
+--+--+--+  +--+--+
|        |  |     |
+--+--+  +--+  +--+
      |        |
+--+--+  +--+  +--+
|        |  |     |
+--+  +--+--+  +--+
   |  |  |     |
   +--+  +--+--+
```

## 数据范围与提示

对于所有数据，$1 ≤ n, m ≤ 100$，且四条边界上都至少有一个空格。
对每个测试点，若连通关系正确但未完全去除多余线段，可获得该测试点 $40\%$ 分数。

# 幸运团队

> 时空限制：1秒 / 128MB

## 题目描述

答题背景

六一儿童节期间，南京博物院面向小学生开展系列活动，每位参观者入场时会收到一张
卡片，每张卡上注明入园编号作为参观者的唯一性标志。

“幸运团队”可以自由组队，通过资格认定所有成员均有一份神秘礼品。
若某团队共有 $n$（$n>0$）位同学参加游戏，则先将这 $n$ 位同学按入园编号从小到大排序，
然后随机分发幸运号，接下来计算相邻同学间的幸运号差值。

若运算得到的结果集恰好是{$1, 2,…,n-1$}，则这个团队就是“幸运团队”。

例如，幸运数为 1 4 2 3 的 4 人团队就是“幸运团队”，因为其幸运号差值分别为 3,2,1。 

请写一个程序判定前来参与活动的团队是否是“幸运团队”。

## 输入格式

第一行，包含一个整数 $T$（$0<T \leq 10$），表示团队的个数。

接下来，每个团队包含一行信息：

第一个数 $n$ 为序列长度，接下来有 $n$ 个正整数，依次表示每个同学的幸运号。

## 输出格式

每个团队输出一行，若该团队是“幸运团队”，则输出"$Lucky$"，否则输出"$Thanks$"。



## 输入输出样例

```input
2
4 1 4 2 3 
4 2 3 4 1
```

```output
Lucky
Thanks
```

## 数据范围与提示

$100\%$的测试数据， $1 < T \leq 10 $  ,$0< n \leq 3000 $ ，每个同学幸运号都在 $1..n$ 之间。

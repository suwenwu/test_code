# 场馆推荐

> 时空限制：1秒 / 128MB

## 题目描述

在博物院入口，会对所有同学进行信息采集，将同学们感兴趣的项目记录到大数据平台，然后根据同学的兴趣爱好，进行场馆推荐。

如果某同学的编号为正整数 $n$，数据平台会挑选出编号为 $n$ 因子的所有同学感兴趣的项目及场馆信息罗列到电子屏幕。

同学们很想比一比谁被推荐的项目最多。

## 输入格式

一行，包含一个正整数 $n$，表示某同学的入园编号。

## 输出格式

一行，包含一个数，表示电子屏中展示的信息中相关学生的数目。

## 输入输出样例

```input
10
```

```output
4
```

## 数据范围与提示

对于 $100\%$ 的测试数据，有 $1<n<1 000 000 000$。

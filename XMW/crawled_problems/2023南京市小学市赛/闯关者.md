# 闯关者

> 时空限制：1秒 / 128MB

## 题目描述

$B$ 组在玩拼图闯关。$n$ 位同学按 $1...n$ 编号顺时针方向围成一圈玩拼图。

第一轮由 $1$ 号同学拼图。对于之后的每一轮，假设上一轮喊出的数字是 $x$，则从上一轮喊出数字的下一位同学开始，沿着顺时针方向数 $x$ 个同学。如果这个同学手中没有卡片，则继续顺时针方向数数，直到遇到第一个有卡片的同学为止。

## 输入格式

一个正整数 $n$，表示小组人数。

## 输出格式

一个整数，表示最后一位完成拼图的同学编号。

## 输入输出样例

```input
5
```

```output
5
```

## 数据范围与提示

输入的数据保证小于 $100$。

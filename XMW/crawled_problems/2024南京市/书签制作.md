# 【2024南京市信息与未来】书签制作

> 时空限制：1秒 / 128MB

## 题目描述

为了激励兴趣⼩组的同学 ，⽼师想根据同学们的每⽉做题情况制作书签 。例如 ，ABC 三位同学做题情况如下：$A$ 同学 $8$ 题， $B$ 同学 $5$ 题 ，$C$ 同学 $3$ 题 ，若制作书签， 内容则为 “$1-2-3$”， 即 $A$ 第⼀ 、 $B$ 第⼆ 、$C$ 第三 。⽼师发现 ，针对 $A、B、C$ 的做题情况 ，最多可制作出 $13$ 种书签 。

![图片](/file/2024/05/20240513022120_书签.png)

兴趣⼩组共有 $n$ 名学⽣ ，那么 ，⽼师最多可能制作出多少种书签？

## 输入格式

输⼊⼀⾏ ，⼀个正整数 $n$。

## 输出格式

输出⼀⾏ ，⼀个整数 ，表⽰不同书签的数量。

## 输入输出样例

### 样例 #1

```input
3
```

```output
13
```

### 样例 #1

```input
17
```

```output
130370767029135901
```

## 数据范围与提示

- 对于 $50\%$ 的数据 ，$n ≤ 10$。
- 对于 $100\%$ 的数据， $1 ≤ n ≤ 17$。
你需要 long long 数据类型储存最终结果。

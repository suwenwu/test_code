# 【2024南京市信息与未来】中位数

> 时空限制：1秒 / 128MB

## 题目描述

在⼀个由不同的数字组成的集合中， 中位数是⼀个元素 $m$ ，它满⾜⼤于 $m$ 的元素数量等于⼩于 $m$ 的元素数量 。

例如：
- 集合 {$1, 4, 2, 5, 7$} 中位数是 $4$， 因为两个元素 ($5$ 和 $7$) ⼤于 $4$，⽽ $2$ 个元素 ($1$ 和 $2$) ⼩于 $4$。
- 集合 {$1, 5, 8, 3$} 没有中位数， 因为它没有任何⼀个元素满⾜上述定义。
给定⼀个整数集合 ，如果其中有中位数 ，则返回该中位数 ，否则返回 $−1$ 。

## 输入格式

输⼊第⼀⾏⼀个整数 $n$ ，表⽰集合的⼤⼩。
输⼊第⼆⾏ $n$ 个正整数 ，表⽰集合中的数。

## 输出格式

输出⼀⾏⼀个中位数 ，如果没有 ，输出 $−1$。

## 输入输出样例

### 样例 #1

```input
5
1 4 2 5 7
```

```output
4
```

### 样例 #1

```input
4
1 5 8 3
```

```output
-1
```

### 样例 #1

```input
9
66 53 47 86 18 21 97 92 15
```

```output
53
```

## 数据范围与提示

对于 $100\%$ 的数据 ，满⾜ $n ≤ 50$ ，且集合中的元素不超过 $100$。

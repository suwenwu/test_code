# 【2024南京市信息与未来】乐乐玩木棒

> 时空限制：1秒 / 128MB

## 题目描述

乐乐有⼀些⼩⽊棒 ，经过多天的摸索 ，他已经知道拼出 $0, 1, 2, … 9$ 这 $10$ 个数字需要的⼩⽊棒数量。

乐乐有 $n$ 根⼩⽊棒 ，他要拼出⼀个不含前导 $0$ 的整数 。乐乐希望拼出⼀个最⼤的数，请你帮他编程计算。
⽊棒不需要全部⽤完。

## 输入格式

输⼊第⼀⾏ $10$ 个整数 ，分别表⽰ $0, 1, 2, … , 9$ 这些数字需要的⼩⽊棒数量。
输⼊第⼆⾏⼀个整数 $n$ ，表⽰乐乐拥有的⼩⽊棒总数 。输⼊数据满⾜⾄少能拼出⼀个⾮ $0$ 的数。

## 输出格式

输出⼀⾏ ，为 $n$ 个⼩⽊棒能拼出的最⼤数字。

## 输入输出样例

### 样例 #1

```input
6 7 8 22 22 22 22 22 22 22
21
```

```output
210
```

### 样例 #1

```input
1 7 8 22 22 22 22 22 22 22
21
```

```output
100000000000000
```

## 数据范围与提示

- 对于 $40\%$ 的数据 ，满⾜最终答案不超过 $1, 000, 000$。
- 对于 $100\%$ 的数据 ，满⾜ $1 ≤ n, a_i ≤ 10, 000$。

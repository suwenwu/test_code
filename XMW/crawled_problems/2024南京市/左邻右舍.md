# 【2024南京市信息与未来】左邻右舍

> 时空限制：1秒 / 128MB

## 题目描述

编号为 $1, 2, … , n$ 的 $n$ 个⼩朋友按顺序围成⼀个圈 ，编号为 $x$ 的同学想知道他左邻右舍分别是谁。

![图片](/file/2024/05/20240513015724_左邻右舍.jpg)

## 输入格式

输⼊⼀⾏ ，两个整数$n, x,$ 中间⽤空格隔开。

## 输出格式

输出⼀⾏，两个整数，按从⼩到⼤的顺序输出 $x$ 的左邻右舍 ，两数中间⽤⼀个空格隔开。

## 输入输出样例

### 样例 #1

```input
10 3
```

```output
2 4
```

### 样例 #1

```input
3 1
```

```output
2 3
```

## 数据范围与提示

对于 $100\%$ 的数据 ，满⾜ $3 ≤ n ≤ 100$ 且 $x$ 合理。

# 暗号

> 时空限制：1秒 / 128MB

## 题目描述

给定一个等差数列 ${a_i}$ 的首项，第二项，末项，求和。

## 输入格式

输入三个数，分别为等差数列的首项，第二项和末项。

## 输出格式

如果该等差数列只有三项，输出 $a_1+a_2+a_3= s$。

超过三项只显示前两项和最后一项，中间用 ... 省略。

等号左侧的加数如果是负数要加()。

## 输入输出样例

```input
1 2 3
```

```output
1+2+3=6
```

```input
-3 -2 0
```

```output
(-3)+(-2)+...+0=-6
```

## 数据范围与提示

$-10^5<=a_i<=10^5$

$3<=等差数列的项数n<=10^4$

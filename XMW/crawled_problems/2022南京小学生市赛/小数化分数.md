# 小数化分数

> 时空限制：1秒 / 128MB

## 题目描述

给定一个小数，求它的最简分数表示。

## 输入格式

输入 $a.b(c)$，表示整数部分为 $a$，小数不循环部分为 $b$，循环部分为 $c$。

## 输出格式

该小数的最简分数。

## 输入输出样例

```input
0.125

```

```output
1/8

```

```input
0.(3)
```

```output
1/3
```

```input
3.(142857)
```

```output
22/7
```

## 数据范围与提示

$100\%$ 的小数不超过 $15$ 位

原有$1$ 个长度 $52$ 位数据已经被移除

# 【2025南京市信息与未来】九头鸟和九尾鸟

> 时空限制：1秒 / 128MB

## 题目描述

小铭在看一本神话传说，“在一片神秘的森林中，住着九头鸟 (九头一尾) 和九尾鸟 (九尾一头)，据说有人可以清楚地看到两种鸟的头和尾”，请帮忙统计九头鸟的个数。

## 输入格式

一行，包含两个整数 $h$ 和 $t$，分别表示两种鸟的头和尾总数。

## 输出格式

一行，包含一个整数，表示九头鸟的个数。

## 输入输出样例

```input
29 21
```

```output
3
```

**说明：** $3 只九头鸟和 2 只九尾鸟共有 29 (27 + 2) 个头和 21 (18 + 3) 个尾。$

```input
381 469
```

```output
37
```

**说明：** $37 只九头鸟和 48 只九尾鸟共有 381 (37 × 9 + 48) 个头和 469 (48 × 9 + 37) 个尾。$

## 数据范围与提示

所有数据满足 $0 < h,t < 10^6$ ，且保证合法 (即 $h,t$ 保证由整数个九头鸟和九尾鸟观察而得)。

# 【2025南京市信息与未来】迷宫生成

> 时空限制：1秒 / 128MB

## 题目描述

小铭今天编程实现了迷宫找路径的算法：在一个 $n$ 行、$m$ 列的迷宫中，有代表空地的格子（用 “.” 表示）和代表障碍的格子（用 “#” 表示）。小铭希望你用程序生成一个 $n$ 行、$m$ 列的迷宫，且从左上角到右下角最少移动步数恰好是 $k$，帮助他测试程序是否正确。

## 输入格式

输入一行三个整数 $n, m$ 和 $k$。

## 输出格式

输出 $n$ 行，每行 $m$ 个字符（“.” 或 “#”），表示迷宫。左上角和右下角必须是空白（“.”）。

## 输入输出样例

```input
4 4 6
```

```output
....
#...
##..
###.
```

```input
3 5 10
```

```output
.#...
.#.#.
...#.
```

## 数据范围与提示

对于 $50\%$ 的数据，满足 $n, m \leq 8$；
对于 $100\%$ 的数据，满足 $n, m \leq 16$。
